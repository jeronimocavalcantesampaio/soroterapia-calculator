# soroterapia-calculator
Preparei um código em PHP pronto para você copiar. Ele não só calcula o gotejamento, mas também tem um "Sistema de Alerta Clínico" baseado nas comorbidades.
Descrição: Projeto desenvolvido para auxiliar farmacêuticos clínicos no cálculo de gotejamento e segurança na prescrição de injetáveis.
Diferente de calculadoras comuns, esta ferramenta aplica lógica clínica para emitir alertas automáticos em pacientes hipertensos ou renais.

Stack: PHP, HTML5, CSS3. Autor: Dr. Jerônimo (Farmacêutico e Desenvolvedor).
